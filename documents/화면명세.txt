네비게이션바 : 1. 홈,  2. 내모임,  3. 포스트,  4. 더보기

1. 홈화면
헤더 : 왼쪽 로고&트리니티 이름,  오른쪽 검색, 알림버튼
메인 :
  상단 모임 타입 링크(일반클럽, 전문클럽, 라운지, 연합) - 링크 클릭시 비슷한 디자인의 상세 모임 리스트 페이지로 이동
  중간부터 하단까지 : 전체 모임 리스트

1-1. 모임 상세 리스트 페이지
헤더 : 왼쪽 뒤로버튼, 중간 : 로고&트리니티,  오른쪽 검색, 알림버튼
메인 :
  상단 : 모임 빅카테고리 탭메뉴(스포츠, 재테크, 음악, 독서) - 메뉴 클릭시 해당 빅카테고리 리스트만 필터링
  중간부터 하단까지 : 선택한 타입의 모든 모임 리스트

1-2. 상세 모임 페이지
헤더 : 왼쪽 뒤로가기버튼, 중간 : 모임이름, 오른쪽 검색, 멤버버튼
메인 :
  상단부터 중간까지 : 왼쪽 - 커다란 프로필사진, 중앙~오른쪽 - 정보표시(이름, 인원수, 생성일, 카테고리 등)
  프로필사진 아래 : 대표 사진들 정렬
  대표사진들 정렬 아래 : 메뉴링크리스트(게시판, 사진첩, 캘린더 등) - 클릭시 해당 메뉴 상세페이지로 이동(단순 리스트 페이지) )
  우측 하단 고정 : 글쓰기 버튼 고정 - 클릭 시 글쓰기 페이지 이동
  하단으로 스크롤 : 각 게시판별 포스트 몇 개씩 출력, 중간중간 광고삽입


2. 내모임
헤더 : 왼쪽 로고&트리니티 이름,  오른쪽 검색, 알림버튼
메인 :
  상단 모임 타입 탭메뉴(일반클럽, 전문클럽, 라운지, 연합) - 클릭시 해당 메뉴 데이터로 필터링
  중간부터 하단까지 : 가입한 전체 모임 리스트, 즐겨찾기한 모임은 상단 노출
    * 모임리스트안에는 기본정보와 3개 정도의 최신/핫 게시물도 출력하기
  우측 하단 고정 : 달력모양버튼 - 클릭시 유저 전용 캘린더 페이지 표시
  스크롤 : 내모임리스트를 지나면 추천모임리스트와 광고노출


3. 포스트
헤더 : 왼쪽 로고&트리니티 이름,  오른쪽 검색, 알림버튼
메인 :
  상단 : 포스트 카테고리 탭메뉴(컬럼, 뉴스, 구인구직, 홍보) - 링크 클릭시 비슷한 디자인의 상세 모임 리스트 페이지로 이동
  중간부터 하단까지 : 전체 포스트 리스트 - 클릭시 상세페이지 이동
    * 포스트 상세페이지는 네이버카페 상세페이지 동일
  우측 하단 고정 : 포스트쓰기버튼 - 클릭시 작성페이지로 이동


4. 더보기
  상단부터 중간까지 : 유저프로필정보 - 기본정보외에도 가입한모임개수, 유저등급, 자기소개 표시
  중간부터 무한스크롤 : 제휴, 광고, 자체판매상품홍보, 모임광고/홍보 등 내용 표시
  우측 하단 고정 : 설정버튼
    * 설정은 단순 수직 리스트로, 앱설정, 공지사항, 고객문의, 회사소개, 앱버전정보/저작권정보, 최하단에 회원탈퇴버튼
